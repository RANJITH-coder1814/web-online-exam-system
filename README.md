# web-online-exam-system
An Online Examination System built using HTML, CSS, JavaScript, and backend integration to conduct online tests securely. The system allows students to take exams, calculates scores automatically, and provides instant results.

An Online Examination System built using HTML, CSS, and JavaScript. 
This project allows students to attempt multiple-choice questions and get instant results after submission.

## 🚀 Features

- Multiple choice questions
- Automatic score calculation
- Simple and clean UI
- Beginner-friendly frontend project
- Responsive layout

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

online-examination-system/

## 📦 How to Run

1. Download or clone the repository
2. Open index.html in your browser
3. Attempt the exam and click Submit
